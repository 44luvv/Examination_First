# Задание 1.
Реализовать не менее 5 алгоритмов сортировки, обязательно должна быть реализована
quick sort. Сравнить скорость сортировки больших и маленьких списков данных, наглядно
продемонстрировать скорости работы сортировок на больших и маленьких списках данных,
сделать вывод какая сортировка и для чего лучше подходит.
# Задание 2.
Реализовать телеграм-бота на свободную тему. Бот обязательно должен уметь:
отвечать на команды
реагировать на кнопки
реагировать на ввод просто текста
уметь работать хотя бы с каким-то из видов контента - картинки, аудио, видео, что
угодно.
